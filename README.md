# Edward Amann Portfolio Site

This is a static portfolio starter for GitHub Pages.

## What to Edit

- Check that the GitHub link matches your exact GitHub username.
- Add an email address or LinkedIn link if you want direct contact options.
- Add more project cards when you publish new games.

## Publish With GitHub Pages

1. Create a new GitHub repository.
2. Upload everything inside this `portfolio-site` folder.
3. In GitHub, open the repository settings.
4. Go to Pages.
5. Set the source to the `main` branch and `/root`.
6. Save, then wait for GitHub to show your Pages URL.

Your homepage file must stay named `index.html`.
